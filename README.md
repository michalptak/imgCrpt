# imgCrpt
AES CBC image encryption GUI. 
Works with .bmp files.

### Prerequisites


- Python 3.x

- Pillow

- pycryptodomex

- PyQt5

### How to run


0. (optional) Setup virtual env.
1. `pip install arcade pillow pycryptodomex pyqt5`
2. Run the script: `python imgCrpt.py`
